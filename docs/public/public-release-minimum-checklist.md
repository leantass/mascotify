# Checklist minima de salida publica de Mascotify

## Base legal y publica minima

- Politica de privacidad publicada
- Canal real de soporte publicado
- Canal para consultas de privacidad publicado
- Descripcion publica consistente entre web y stores

## Web

- URL publica definitiva
- Pagina o documento publico de privacidad
- Pagina o documento publico de soporte
- Contacto visible en la entrega final

## Mobile / stores

- Politica de privacidad lista para enlazar
- Soporte o contacto listo para enlazar
- Metadata final de store
- Screenshots y piezas visuales finales

## Lo que hoy ya puede salir del repo

- borrador minimo de politica de privacidad
- borrador minimo de soporte y contacto
- base para adaptar esos textos a web y stores

## Lo que depende de publicacion externa

- dominio o URL final
- correo o formulario real de contacto
- responsable visible
- links finales para Play Store y App Store
